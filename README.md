# waybar pimp-module

[PimpMyRice](https://github.com/daddodev/pimpmyrice) module for [Waybar](https://github.com/Alexays/Waybar).

## Set up

```bash
pimp clone module waybar
```

add to your style:

```json
{
    ...
    "modules_styles": {
        "waybar": {
            "variant": "split", # or "solid"
            "colorful": "no" # or "background" or "foreground"
        }
    }
}
```
